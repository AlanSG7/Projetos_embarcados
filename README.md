# Projetos_embarcados
Repositório de projetos focados na utilização de microcontroladores e integração de sensores. 

Os projetos desse repositório foram criados, especialmente, ao longo do programa Embarcatech, projeto de residência tecnológica apoiado pelo MCTI/Brasil. O programa utiliza como base a plataforma BitDogLab, a qual utiliza o microcontrolador RP2040.

Todos os códigos foram escritos utilizando o Pico SDK e outras bibliotecas convenientes (como a lwIP para conectividade e a FreeRTOS para implementação de sistema operacional de tempo real).

Aqui, destaca-se a pasta "projeto_final_Internet_of_Medical_Things", projeto final da residência tecnológica Embarcatech, no qual foi desenvolvida uma aplicação utilizando FreeRTOS, conectividade (Wi-Fi e MQTT), além da integração de múltiplos módulos (utilizando protocolos de comunicação comunicação SPI, I2C e UART).
